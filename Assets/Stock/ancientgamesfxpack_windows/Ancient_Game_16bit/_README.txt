Description
-----------

Ancient Game is a sound effects library created and produced by Epic Stock Media. The library presents a diverse set of audio assets that are particularly useful to game and software developers, sound designers and producers. 

We encourage you to offer comments and make suggestions, ask questions, or tell us what you'd like to see in new products. Email: info@epicstockmedia.com. We appreciate your feedback. 


Technical Requirements
----------------------

Playback of files can be achieved with modern digital audio workstations, game and software development platforms and with any device that can process the included 16bit 44.1k format audio files.
